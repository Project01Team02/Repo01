This is a website that uses two APIs to find out what your drink is based on what movie you are watching. The cocktail API has a database of over 635 drinks. The movie API obtains information from user inputted database on content of the movie. From there, we match the movie with a drink that has been randomized! We hope you enjoy and drink responsibly.
